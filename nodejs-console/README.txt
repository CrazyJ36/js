This is how to run javascript in any system/terminal with nodejs installed.

Install node to run javascript on console:
- Windows:
Download installer node.js from webpage:
https://nodejs.org/en/download/

- Linux:
in terminal, install package nodejs, it's ~16mb size:
$ sudo apt-get install nodejs

# Code Example:
--------
console.log("hello world")
--------

Compile/Run:
- node file.js
